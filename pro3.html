TNSWC Website

This website serves as a comprehensive resource on women's rights, relevant laws, and legal aid. It offers downloadable materials, public awareness pamphlets, and publications, along with multimedia content to educate the public on legislation concerning women. Additionally, the site features a regularly updated section with news, events, and achievements of the commission. The website also includes Tamil translation for the entire page, a dark mode option, and font resizing capabilities. Explore the user journey down below.

TNAWB Website

This website acts as a guide for the Tamil Nadu Animal Welfare Board and relevant laws. It provides books, online materials, and multimedia information to educate the public about animal legislation. The website offers the options to change the font size, go to dark mode, and translate every page into Tamil. It also features an updated section that features the commission's structure, news, operations, and achievements on an ongoing basis.
